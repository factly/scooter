# Output

####

The editor API contains a number of methods and commands which can be used to
interact with the editor instance. This section will cover the most important
ones.

<br />

### **Use EditorContent component**

```js
import { EditorContent } from "@factly/scooter-core";
```

### **Include CSS in your project**

```html
<link
  href="https://cdn.jsdelivr.net/gh/factly/scooter@master/dist/editor-content.min.css"
  rel="stylesheet"
/>
```

Use the above CDN link to include the styles in your project. Add the class
**scooter-editor-content** to the wrapper of the output content.
