# Override defaults

By default the following the optons are available:

- `font-color`
- `bold`
- `italics`
- `underline`
- `strike`
- `link`
- `paragraph`
- `h1`
- `bulleted-list`
- `numbered-list`

You can edit the available options by passing an array of strings to the
`defaults` prop. In addition to the above options you can also pass the
following options: `h3`, `h4`, `h5`, and `h6`.

<Canvas>
